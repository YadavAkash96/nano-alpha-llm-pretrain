# Phase 6: Reporting and Cross-Run Comparison

## Definition
Reporting consolidates checkpoint and run artifacts into decision-ready summaries.

## What We Implement
1. Checkpoint-vs-metric master table
2. Best-checkpoint selection per metric/task
3. Plot pack: CE/PPL/ECE/benchmark curves
4. Run card with data, hyperparameters, environment, and outcomes

## Why It Matters
1. Supports reproducible selection decisions.
2. Enables transparent communication to collaborators.
3. Makes future run comparisons straightforward.

## What It Evaluates
1. Relative model quality and calibration
2. Reproducibility and traceability
3. Practical readiness for deployment or continued training
